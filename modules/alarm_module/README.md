# description
An alarm snooze/stop button and a display to show the get out of home time and whatever information you want, like weather, and traffic

## components
- resistors:
    + 1x 220立
    + 2x 510立
    + 4x 1k立
    + 1x 10k立
- capacitors:
    + 2x 22pf ceramic capacitor
- transistors:
    + 1x BD138
    + 1x BC547
- microcontrollers:
    + 1x ATmega328P
    + 1x ESP8266 (ESP-01) 512K/1MB
- 1x lcd display (I use the lmb162abc)
- 1x LM1117T 3.3V regulator
- 1x 16Mhz crystal oscillator
- 1x 28 pin dip socket
- 1x 2x4 pin socket
    
## extra consumables
- 1x 5V power supply
- dot matrix PCB
- wires
- solder

## tools
- soldering iron
- Multimeter
- USB to TTL converter
- an arduino uno
